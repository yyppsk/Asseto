Version 1 and Version 3 backup
==============================

These files were removed from the active Vite app after Version 2 became the single live track experience.

- `src/trackData.js`, `src/scene/track.jsx`, `src/scene/environment.jsx`, and `src/scene/placements.json` hold the procedural Version 1 track and placement layer.
- `src/scene/realTrack-with-version3.jsx` is a copy of the real-track loader before the active Version 3 config was removed.
- `public/data` contains the TUM Spa CSV data used by Version 1.
- `public/models/real track 2` contains the Version 3 test GLB.

Nothing in the active app imports this folder.
